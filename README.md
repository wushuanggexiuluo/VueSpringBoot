# Vue+SpringBoot前后端分离个人博客项目

#### 介绍
个人练手的Vue+SpringBoot前后端分离个人博客项目，比较简单适合小白(适配双端)

#### 版本说明：
Vue2+Redis5.8+Mysql5.8+Springboot2.7
